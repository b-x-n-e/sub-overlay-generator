OBS Setup

  1. Click Generate & Download HTML above.
  2. In OBS, click the + under Sources → Browser.
  3. Check "Local file" and browse to the downloaded .html file.
  4. Set width & height to your stream resolution (e.g. 1920 × 1080).
  5. Done — the overlay is transparent and will show alerts automatically.

The HTML file contains everything (media, audio, code). No internet needed for the overlay once downloaded, except for the Twitch chat connection.
