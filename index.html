OBS Setup

  1. Click Generate & Download HTML above.
  2. In OBS, click the + under Sources → Browser.
  3. If you only use sub alerts (no follow): check "Local file" and browse to the .html file.
  4. If you use follow alerts: run npx serve /path/to/folder in a terminal, then paste the URL (e.g. http://localhost:3000/overlay.html) into the OBS Browser source URL field.
  5. Set width & height to your stream resolution (e.g. 1920 × 1080).
  6. Done — the overlay is transparent and will show alerts automatically.

Follow alerts require the overlay to be served over HTTP (not loaded as a local file) because the Twitch API blocks requests from file:// origins. Sub-only overlays work fine as local files.
